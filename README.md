# Nuxt 3 Minimal Starter

Look at the [Nuxt 3 documentation](https://nuxt.com/docs/getting-started/introduction) to learn more.

## Development

## BE

Run Docker desktop

Run Docker containers

```bash
  yarn docker
```

Run Prisma

```bash
  yarn push
```

```bash
  yarn migrate
```

Run project

```bash
  yarn dev
```

Bonus

View database

```bash
  yarn studio
```

## FE

```bash
yarn dev

```
